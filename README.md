# 🏛️ 南京风景名胜介绍 / 난징 관광명소 소개

## 🇨🇳 中文

本项目是一款基于 Flutter 开发的中韩双语旅游导览应用，旨在向国内外游客介绍南京丰富的历史文化遗产、著名旅游景点、特色美食以及城市风貌，为游客提供便捷、直观、智能的旅游信息服务。

## 🇰🇷 한국어

본 프로젝트는 Flutter 기반의 중·한 이중 언어 관광 안내 애플리케이션입니다. 난징의 풍부한 역사문화유산, 대표 관광명소, 지역 특색 음식 및 도시 문화를 소개하며 관광객에게 편리하고 직관적인 관광 정보 서비스를 제공합니다.

# ✨ 主要功能 / 주요 기능

## 🇨🇳 中文

### 🏛️ 景点介绍

展示南京著名旅游景点的历史背景、文化价值及旅游信息。

包括：

* 中山陵
* 明孝陵
* 夫子庙
* 南京总统府
* 玄武湖
* 鸡鸣寺
* 老门东
* 南京博物院

### 🍜 美食推荐

介绍南京特色美食：

* 南京盐水鸭
* 鸭血粉丝汤
* 牛肉锅贴
* 小笼包
* 梅花糕
* 桂花糖芋苗

### 🌐 中韩双语切换

支持中文与韩文自由切换。

### 📱 响应式界面

适配不同尺寸移动设备。

### 📷 图片展示

通过高清图片展示南京旅游资源。

## 🇰🇷 한국어

### 🏛️ 관광명소 소개

난징의 대표 관광명소에 대한 역사적 배경과 문화적 가치를 제공합니다.

포함 관광지：

* 중산릉
* 명효릉
* 부자묘
* 난징 총통부
* 현무호
* 계명사
* 노문동
* 난징박물원

### 🍜 음식 추천

난징의 대표 음식을 소개합니다.

* 난징 염수오리
* 오리피 당면탕
* 소고기 군만두
* 샤오롱바오
* 매화떡
* 계화 토란 디저트

### 🌐 중·한 이중 언어 지원

중국어와 한국어를 자유롭게 전환할 수 있습니다.

### 📱 반응형 UI

다양한 모바일 기기를 지원합니다.

### 📷 이미지 갤러리

고화질 이미지를 통해 관광 정보를 제공합니다.
